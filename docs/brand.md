# Balcu Apps — Sistema de marca

Identidad visual compartida del ecosistema. Logos en `public/brand/`.

## Jerarquía

```text
Balcu Apps (paraguas)
├── Balcu Hub (portal)
├── Smart Control Security (referencia en hub — app de terceros, sin cambios de imagen allí)
├── RotaTuDisfraz
└── Chacachon Stories
```

## Reglas

1. Paraguas en hub y docs cross-app.
2. Sub-marca dentro de cada producto propio (RotaTuDisfraz, Chacachon).
3. No mezclar isotipos entre apps.
4. Smart Control: referencia visual en hub únicamente.

Copia canónica del workspace: `../brand/` (fuera de este repo Git).
